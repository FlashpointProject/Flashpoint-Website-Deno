3) **Saving Space**. This is in 2 parts:
- **Pick and choose software**: The installer (and manager openable in the bottom left of the launcher) lets you choose which software you want installed. Don't play Java / Unity / Shockwave games? Just turn off their support packs! Even if you don't install a support pack these games will still show in results and the launcher will make sure to download them if you try to launch any of the games from which you're missing any.
- **Logos and Screenshots**. On-demand images now download compressed copies by default. This uses roughly 80% less space for no noticeable reduction in quality. If you still want uncompressed images you can disable compression on the Config page then use the clear images button to make sure they download in full resolution the next time they're required.
## 📝  I'm still reading by this point, what else changed?
- **For Curators**. Flashpoint 12 comes with a lot of improvements to the old curate tab. I'm not going to go into detail here but there's lots of new things to play with to curate new games, submit changes for existing games and edit live metadata from games from inside the launcher itself. Extensions also have new ways to interact with curations so if you want to add something please take a look, and don't be afraid to ask for new endpoints if you can't achieve your goal with the existing ones.

- **FPFSS**. People that pay attention will know that our Flashpoint Fantastic Submission System has been an integral part of people submitting their curations and having them quality checked before inclusion in the archive. FPFSS has grown to be the backbone of most of the new changes in version 12. It is now responsible for serving *all* live information about the entire archive; Most changes to games and tags are submitted through this system in one way of another.
